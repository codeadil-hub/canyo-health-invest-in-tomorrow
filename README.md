# Canyo Health Invest in Tomorrow

A modern investor-focused web application built using React and Vite.

## Overview

This project was developed to create a responsive and user-friendly platform for presenting investment and healthcare-related information.

## Technologies Used

- React
- JavaScript
- Vite
- HTML
- CSS

## Features

- Responsive design
- Modern UI
- Component-based architecture
- Fast performance using Vite

## What I Learned

Through this project, I gained experience in:
- Building React applications
- Organizing frontend project structure
- Working with reusable components
- Creating responsive user interfaces

## Getting Started

```bash
npm install
npm run dev
```
